# Evasion Advanced Browser

An Electron-based advanced desktop browser with gaming controls, workspaces, tab groups, extensions, privacy settings and an encrypted password vault.

## Run

```bash
npm install
npm start
```

Open the browser menu and choose **GX Gaming Hub** for performance controls, game launchers, screenshots, reader mode, notes and workspaces.

## Development safety
Use test credentials while developing the password manager. This project is educational and has not received an independent security audit.

## GX Homepage and Incognito update
- New neon gaming start page with gaming launchers, GX status panel, focus mode and gaming discovery cards.
- Incognito windows now use an isolated temporary Electron session, a dedicated purple toolbar identity and a private start page.
- Incognito home searches use DuckDuckGo by default and normal session tabs are not restored in private windows.


## Z+ Security Center
Evasion includes transparent ad/tracker/miner filtering, Global Privacy Control, selected anti-fingerprinting protections and permission restrictions. It does not claim to be undetectable or provide anonymity against governments, networks or websites. For anonymity, use established audited tools and understand their limits.
