# Evasion Browser Privacy Notice

Evasion Browser stores browser profile information, settings, history,
bookmarks, downloads metadata, workspace data, and the encrypted password vault
locally on the device unless the user explicitly configures an external sync or
AI service.

## Local information

- Profile name and preferences
- Browsing history and bookmarks
- Download records
- Workspace, theme, and tab-session information
- Encrypted password-vault data

## Optional external services

Features such as online AI, WebDAV sync, third-party search, websites opened by
the user, and compatible extensions may communicate with their own providers.
Their privacy policies apply independently.

## Private windows

Private windows use temporary Electron sessions and are intended to avoid
saving normal-session history and cookies. Private browsing does not make a
user invisible to websites, schools, employers, internet providers, or network
administrators.

## Password vault

The vault is encrypted locally. Users should test the software with noncritical
credentials until the application has received an independent security review.
